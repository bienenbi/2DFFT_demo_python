二次元逆フーリエ変換による画像再構成デモ

######################################################
		概要

このコードは,二次元フーリエ変換を行った後に振幅スペクトルをプロットし,各画素に対して逆フーリエ変換を行うことによって画像を再構成する過程を体験できるデモである.
  このコードはPythonでコーディングされている. また,このコードを実行するにあたって,

・PIL（Python Imaging Library）
・Numpy
・Matplotlib

をpipコマンドでインストールしておくこと.


######################################################
		使用方法

　FFT.py と同じ階層のフォルダ内に,「Fourier.jpg」という画像ファイルを用意しておくこと.
その後実行すると,左上に元画像,真中上に振幅スペクトルの画像が表示される.
  その状態で,真中下の画像をクリックすると,右下にその点での空間領域での波形が表示され,左下に再構成した空間領域での画像が表示される.
また,サイズの大きな画像を利用すると横幅256pxになるまで縮小されるため注意すること.

***注意***
  1画素づつ計算する仕様上非常に計算速度が遅くなっているため,対応策としてCtrl+左クリックによって微小な空間の波形がまとめて再構成される.
詳しくはデモを参照すること.
　また,再構成済みの周波数領域に対して右クリックを行うと,その点の波形を再構成される前の状態に戻すことができる.
低周波領域の消去などに利用すること.
  Maxwidthの値を変更すると画像の大きさを変更できるため,速度が遅いときには変更すること.
